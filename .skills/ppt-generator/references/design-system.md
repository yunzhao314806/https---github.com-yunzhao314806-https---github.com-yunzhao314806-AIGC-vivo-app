## 配色规则（强制执行）

### 严格遵守配色方案

**只使用提供的配色方案，不自行创建或修改颜色。**

- 所有颜色必须来自用户提供的配色方案
- 不得使用方案以外的颜色
- 不得修改配色方案中的颜色（亮度、饱和度、混合）
- **唯一例外**：使用 `transparency` 属性（0-100）添加透明度

```javascript
// ✅ 正确：使用配色方案中的颜色
slide.addShape(pres.shapes.RECTANGLE, { fill: { color: theme.primary } });
slide.addText("标题", { color: theme.accent });

// ❌ 错误：使用方案外的颜色
slide.addShape(pres.shapes.RECTANGLE, { fill: { color: "1a1a2e" } });
```

### 禁用渐变

**禁止使用渐变，只用纯色。**

### 禁用动画

**禁止使用动画和切换效果。** 所有幻灯片必须是静态的。

## 字体参考

### 推荐字体

| 语言 | 默认字体 | 备选字体 |
|----------|-------------|--------------|
| **中文** | 微软雅黑（Microsoft YaHei） | — |
| **英文** | Arial | Georgia、Calibri、Cambria、Trebuchet MS |

- 中英混排内容：中文用微软雅黑，英文用选定字体
- 优先使用系统字体，确保跨平台兼容
- 标题与正文可以使用不同字体搭配（如 Georgia + Calibri）

### 推荐字体搭配

| 标题字体 | 正文字体 |
|-------------|-----------|
| Georgia | Calibri |
| Arial Black | Arial |
| Calibri | Calibri Light |
| Cambria | Calibri |
| Trebuchet MS | Calibri |
| Impact | Arial |
| Palatino | Garamond |
| Consolas | Calibri |

**选择有个性的字体搭配**——不要所有地方都用 Arial。标题字体要有特色，正文字体要简洁易读。

### 正文不加粗

**正文文字和说明/图例文字不得加粗。**

- 正文段落、描述 → 常规字重
- 说明、图例、注脚 → 常规字重
- 加粗仅用于标题和小标题

```javascript
// ✅ 正确
slide.addText("主标题", { bold: true, fontSize: 36, fontFace: "Arial" });
slide.addText("正文内容。", { bold: false, fontSize: 14, fontFace: "Arial" });

// ❌ 错误
slide.addText("正文内容。", { bold: true, fontSize: 14 });
```

## 风格 Recipe

通过调整圆角半径（`rectRadius`）和间距，同一设计可以呈现 4 种不同视觉风格。选择与演示文稿基调匹配的 recipe。

> **单位说明**：PptxGenJS 使用英寸。幻灯片尺寸为 10" x 5.625"（LAYOUT_16x9）。

### 风格总览

| 风格 | 圆角半径 | 间距 | 适合场景 |
|-------|--------------|---------|----------|
| **锐利紧凑** | 0 ~ 0.05" | 紧密 | 数据密集、表格、专业报告 |
| **柔和均衡** | 0.08" ~ 0.12" | 适中 | 企业、商务演示、通用场景 |
| **圆润宽松** | 0.15" ~ 0.25" | 宽松 | 产品介绍、营销、创意展示 |
| **胶囊开阔** | 0.3" ~ 0.5" | 开放 | 品牌展示、发布会、高端演示 |

### 锐利紧凑

**视觉特点**：几何感强，信息密度高，正式严肃。

| 类别 | 值（英寸） | 备注 |
|----------|---------------|-------|
| 圆角 — 小 | 0" | 直角 |
| 圆角 — 中 | 0.03" | 微圆角 |
| 圆角 — 大 | 0.05" | 轻微圆角 |
| 元素内边距 | 0.1" ~ 0.15" | 紧凑 |
| 元素间距 | 0.1" ~ 0.2" | 紧凑 |
| 页面边距 | 0.3" | 窄 |
| 区块间距 | 0.25" ~ 0.35" | 紧凑 |

### 柔和均衡

**视觉特点**：适度圆角，舒适留白，专业又亲切。

| 类别 | 值（英寸） | 备注 |
|----------|---------------|-------|
| 圆角 — 小 | 0.05" | 轻微圆角 |
| 圆角 — 中 | 0.08" | 中等圆角 |
| 圆角 — 大 | 0.12" | 较大圆角 |
| 元素内边距 | 0.15" ~ 0.2" | 适中 |
| 元素间距 | 0.15" ~ 0.25" | 适中 |
| 页面边距 | 0.4" | 标准 |
| 区块间距 | 0.35" ~ 0.5" | 适中 |

### 圆润宽松

**视觉特点**：大圆角，充足留白，友好现代。

| 类别 | 值（英寸） | 备注 |
|----------|---------------|-------|
| 圆角 — 小 | 0.1" | 中等圆角 |
| 圆角 — 中 | 0.15" | 大圆角 |
| 圆角 — 大 | 0.25" | 超大圆角 |
| 元素内边距 | 0.2" ~ 0.3" | 宽松 |
| 元素间距 | 0.25" ~ 0.4" | 宽松 |
| 页面边距 | 0.5" | 宽 |
| 区块间距 | 0.5" ~ 0.7" | 宽松 |

### 胶囊开阔

**视觉特点**：完全胶囊形圆角，大量留白，轻盈开放，品牌感强。

| 类别 | 值（英寸） | 备注 |
|----------|---------------|-------|
| 圆角 — 小 | 0.2" | 大圆角 |
| 圆角 — 中 | 0.3" | 胶囊形 |
| 圆角 — 大 | 0.5" | 完整胶囊 |
| 元素内边距 | 0.25" ~ 0.4" | 开放 |
| 元素间距 | 0.3" ~ 0.5" | 开放 |
| 页面边距 | 0.6" | 宽 |
| 区块间距 | 0.6" ~ 0.9" | 开放 |

### 组件风格对照

| 组件 | 锐利 | 柔和 | 圆润 | 胶囊 |
|-----------|-------|------|---------|------|
| **按钮/标签** | rectRadius: 0 | rectRadius: 0.05 | rectRadius: 0.1 | rectRadius: 0.2 |
| **卡片/容器** | rectRadius: 0.03 | rectRadius: 0.1 | rectRadius: 0.2 | rectRadius: 0.3 |
| **图片容器** | rectRadius: 0 | rectRadius: 0.08 | rectRadius: 0.15 | rectRadius: 0.25 |
| **输入框** | rectRadius: 0 | rectRadius: 0.05 | rectRadius: 0.1 | rectRadius: 0.2 |
| **徽章** | rectRadius: 0.02 | rectRadius: 0.05 | rectRadius: 0.08 | rectRadius: 0.15 |
| **头像框** | rectRadius: 0 | rectRadius: 0.1 | rectRadius: 0.2 | rectRadius: 0.5（圆形） |

#### PptxGenJS 圆角示例

```javascript
// 锐利风格卡片
slide.addShape("rect", {
  x: 0.5, y: 1, w: 4, h: 2.5,
  fill: { color: "F5F5F5" },
  rectRadius: 0.03
});

// 圆润风格卡片
slide.addShape("rect", {
  x: 0.5, y: 1, w: 4, h: 2.5,
  fill: { color: "F5F5F5" },
  rectRadius: 0.2
});

// 胶囊风格按钮（高度 0.4"，rectRadius 0.2" = 完美胶囊）
slide.addShape("rect", {
  x: 3, y: 4, w: 2, h: 0.4,
  fill: { color: "4A90D9" },
  rectRadius: 0.2
});
```

### 混用规则

#### 1. 外层容器圆角 >= 内层元素圆角

```javascript
// ✅ 正确：外层 > 内层
卡片:   rectRadius: 0.2
按钮:   rectRadius: 0.1

// ❌ 错误：内层 > 外层 → 视觉溢出
卡片:   rectRadius: 0.1
按钮:   rectRadius: 0.2
```

#### 2. 信息密度决定间距

| 区域类型 | 推荐风格 |
|-----------|------------------|
| 数据展示区 | 锐利 / 柔和（紧凑间距） |
| 内容浏览区 | 圆润 / 胶囊（宽松间距） |
| 标题区 | 柔和 / 圆润（适中间距） |

#### 3. 圆角与元素高度的关系

| 元素高度 | 锐利 | 柔和 | 圆润 | 胶囊 |
|---------------|-------|------|---------|------|
| 小（< 0.3"） | 0" | 0.03" | 0.08" | 高度/2 |
| 中（0.3"~0.6"） | 0.02" | 0.05" | 0.12" | 高度/2 |
| 大（0.6"~1.2"） | 0.03" | 0.08" | 0.2" | 0.3" |
| 超大（> 1.2"） | 0.05" | 0.12" | 0.25" | 0.4" |

> **胶囊技巧**：完美胶囊形设置 `rectRadius = 元素高度 / 2`
