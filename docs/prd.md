# 需求文档

## 1. 应用概述

### 1.1 应用名称
Agent驱动的岗位智能匹配与能力图谱系统

### 1.2 应用描述
本系统是一款基于AI智能体技术的双向招聘匹配平台,为求职者和企业提供智能化的岗位匹配服务。系统通过AI对话、能力图谱可视化、智能简历解析等功能,实现求职者与企业岗位的精准匹配,提升招聘效率。

## 2. 用户与使用场景

### 2.1 目标用户
- 求职者:需要快速找到匹配岗位的求职人员
- 企业HR:需要高效筛选候选人的招聘人员
- 平台管理员:负责系统运营与维护的管理人员

### 2.2 核心使用场景
- 求职者通过AI对话快速匹配适合岗位
- 企业发布职位并获得智能推荐的候选人
- 双方通过能力图谱直观了解匹配度
- 管理员监控平台运营数据并进行用户管理

### 3.2 求职者端功能

#### 3.2.1 首页
- 个性化职位推荐:基于用户历史行为、浏览记录及偏好设置,展示推荐职位列表
- 快速搜索与筛选:支持关键词搜索,按地区、薪资、行业等条件筛选职位

#### 3.2.2 简历管理
- 简历文件上传:支持Word(.doc/.docx)、PDF(.pdf)、CSV(.csv)等格式简历文件上传,提供拖拽上传和点击上传两种方式,显示文件上传进度
- 简历文件存储:上传的简历文件存储在Supabase Storage中
- 已上传简历列表:展示已上传简历文件列表,包含文件名、格式、大小、上传时间等信息
- 简历文件管理:支持下载、删除已上传的简历文件
- 简历预览:在线查看已上传简历内容
- 简历编辑:支持在线修改简历信息
- 能力发展中心:展示个人能力雷达图和能力树图,可视化呈现技能掌握情况
- 能力模拟:支持拖拽方式模拟能力值变化,查看对匹配度的影响

#### 3.2.3 职位详情与申请
- 职位详情展示:显示职位描述、任职要求、公司简介、工作环境等完整信息
- 智能匹配度分析:展示个人简历与职位要求的匹配度评分及详细分析
- 一键申请:快速提交职位申请
- 职位收藏:收藏感兴趣的职位便于后续查看

#### 3.2.4 AI匹配与对话
- AI智能对话:通过语音或文字与AI助手进行多轮对话,描述求职意向
- 匹配报告展示:查看AI对话记录及生成的匹配报告,包括匹配亮点、待改进点及建议
- 批注与反馈:对匹配结果进行批注和反馈

#### 3.2.5 个人中心
- 个人信息管理:查看和修改基本信息、教育背景、工作经历等
- 发展报告查看:查看个人发展报告与匹配分析报告
- 系统设置:调整界面风格、通知方式等个性化设置
- 用户反馈:提交意见和建议

### 3.3 企业端功能

#### 3.3.1 首页
- 个性化人才推荐:基于发布的职位需求,展示高匹配度候选人列表
- 快速职位发布:通过语音或文字输入职位需求,系统自动生成职位描述
- 职位管理:查看、编辑、删除已发布职位

- 能力图谱对比:查看候选人能力图谱与岗位需求的匹配热力图
- 多候选人对比:同时对比多个候选人的能力图谱

#### 3.3.3 招聘流程管理
- AI面试:发起AI模拟面试,系统自动生成结构化问题
- 面试评估:查看AI生成的多维评估报告,包括沟通能力、专业水平、岗位适配度等

#### 3.3.4 双向沟通

- 语音转文字:支持语音输入自动转换为文字

#### 3.3.5 数据统计

## 4. 业务规则与逻辑

### 4.1 核心匹配机制

#### 4.1.1 匹配计算规则
- 采用双引擎模式:结构化匹配(70%)+ 规则匹配(30%)
- 结构化匹配:基于六大行业能力树模板(互联网/科技、金融/投资等),对齐简历与岗位结构,通过节点权重计算匹配度
- 规则匹配:提取城市、薪资等信息进行量化评分
- 学历惩罚机制:企业可自定义学历不匹配的惩罚系数(0-100%)
- 最终匹配度 = 结构化匹配得分 × 70% + 规则匹配得分 × 30% - 学历惩罚

#### 4.1.2 行业识别与模板加载
- 系统自动识别简历和职位所属行业
- 根据行业类型加载对应的能力树模板
- 支持六大行业:互联网/科技、金融/投资、制造业、教育培训、医疗健康、其他

#### 4.1.3 匹配报告生成
- 生成包含匹配度评分、优势分析、差距建议的详细报告
- 双向展示:求职者端显示适合岗位,企业端显示推荐候选人

### 4.2 AI对话匹配流程
- 用户发起对话,描述需求(求职意向或岗位要求)
- AI助手进行多轮对话,深入理解需求细节
- 系统基于对话内容进行智能解析和结构化数据提取
- 执行匹配计算,生成匹配结果
- 向用户展示匹配报告并提供优化建议

### 4.3 动态反馈优化机制
- 收集用户行为数据(浏览、申请、收藏等)
- 收集用户反馈(满意度评价、批注等)
- 基于数据分析动态调整推荐算法权重
- 持续优化匹配精准度

### 4.4 简历智能解析规则
- 自动识别简历中的关键信息:基本信息、教育背景、工作经历、项目经验、技能标签等
- 提取技能关键词并进行标准化处理
- 根据目标职位提供简历优化建议

## 5. 异常与边界情况

| 场景 | 异常情况 | 处理方式 |
|------|----------|----------|
| 简历上传 | 文件格式不支持 | 提示用户仅支持Word(.doc/.docx)、PDF(.pdf)、CSV(.csv)等格式 |
| 简历上传 | 文件解析失败 | 提示用户检查文件内容或重新上传 |
| 简历上传 | 文件上传失败 | 提示用户检查网络连接并支持重试 |
| AI对话 | 用户输入无法理解 | 引导用户重新描述或提供示例 |
| 匹配计算 | 简历信息不完整 | 提示用户补充必要信息 |
| 匹配计算 | 无匹配结果 | 提示用户调整筛选条件或扩大搜索范围 |
| 职位申请 | 重复申请同一职位 | 提示用户已申请过该职位 |
| 数据加载 | 网络连接失败 | 提示用户检查网络连接并支持重试 |
| 用户登录 | 账号密码错误 | 提示错误信息并限制连续错误次数 |
| 权限访问 | 无权限访问某功能 | 提示权限不足并引导联系管理员 |

1. 求职者可成功上传简历并查看能力图谱可视化展示
2. 求职者可通过AI对话获得职位推荐,匹配度计算准确
3. 求职者可查看职位详情、申请职位、收藏职位
4. 企业可发布职位并获得候选人推荐
5. 企业可查看候选人能力图谱并进行对比分析
6. 企业可发起AI面试并查看评估报告
7. 双方可通过系统进行实时沟通
8. 管理员可进行用户管理、数据维护和运营监控
9. 匹配算法基于用户反馈实现动态优化
10. 系统在vivo手机环境下运行流畅,界面适配良好
11. 所有用户数据传输和存储均经过加密处理
12. 系统响应时间符合性能要求,关键操作在3秒内完成
